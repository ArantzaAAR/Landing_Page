# Landing_Page
# 🌐 Portfolio Web Personal - Lenguajes de Marcas

Este proyecto consiste en el desarrollo de una página web personal, diseñada con HTML y CSS, con el objetivo de mostrar quién soy, qué he aprendido y qué proyectos he desarrollado durante el curso. Se trata de una carta de presentación digital que refleja tanto el contenido técnico aprendido como mi estilo personal.

---

## 👤 Sobre Mí

En esta sección he redactado una breve descripción personal que resume mi perfil, intereses y formación. El objetivo es generar interés y dar una primera impresión positiva a quienes visiten mi página.

- Estudios actuales.
- Motivación en el ámbito de la tecnología.
- Aficiones o habilidades destacables.

---

## 📄 Currículum Vitae (CV)

He diseñado un CV interactivo utilizando HTML y CSS, donde se destacan:

- Formación académica.
- Experiencia profesional (si la hubiera).
- Habilidades técnicas.
- Idiomas y otras competencias.

Se han aplicado estilos atractivos y modernos, basándome en los ejemplos trabajados en clase y en recursos adicionales de diseño web. La estructura ha sido organizada para facilitar la lectura y el impacto visual.

---

## 🧩 Proyectos

Esta sección actúa como una galería interactiva que recopila mis principales trabajos académicos en las distintas materias del curso:

- **Lenguajes de Marcas**: Ejercicios HTML/CSS, páginas estructuradas, formularios.
- **Programación**: Proyectos en Java o Python aplicando lógica y estructuras de datos.
- **Bases de Datos**: Modelos relacionales, procedimientos y scripts SQL.
- **Entornos de Desarrollo**: Documentación técnica y configuración de entornos.
- **Sistemas Informáticos**: Prácticas sobre redes, terminal y comandos del sistema operativo.

Cada proyecto está enlazado o documentado con una breve descripción y, si es posible, una captura o vista previa.

---

## 📬 Contacto

Incluye un **formulario responsivo** creado con HTML y CSS que permite a cualquier visitante ponerse en contacto de forma sencilla. Contiene campos básicos como:

- Nombre
- Correo electrónico
- Asunto
- Mensaje

Se ha estilizado para mantener coherencia visual con el resto del sitio y garantizar una experiencia de usuario intuitiva.

El sitio web ha sido desplegado en GitHub Pages para hacerlo accesible desde cualquier dispositivo con conexión a internet.
